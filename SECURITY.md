# Security Policy

This repository contains demo and proof-of-concept code intended for education and reference. It is **not** designed for production deployment without significant additional review and hardening.

## Reporting a Vulnerability

If you discover a security issue in this code, please email **aslancarlos@gmail.com** with the details. Please do not open a public issue for sensitive security reports; coordinate disclosure privately first.

I aim to acknowledge reports within a few business days on a best-effort basis. There is no commercial SLA on this repo.

## Disclaimer

This is community-maintained sample code. It is not officially supported by any vendor referenced in the project. Use at your own risk.
